
## Ejercicio
```
Desarrollar un programa que permita calcular el área y el volumen de una esfera, un cubo y un tetraedro regular. 

El programa debe desplegar un menú con las opciones:
1. Calcular dimensiones de una Esfera.
2. Calcular dimensiones de un Cubo.
3. Calcular dimensiones de un Tetraedro Regular.
4. Salir.
```
